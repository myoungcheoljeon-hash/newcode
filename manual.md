# 🤖 네이버 카페 자동화 봇 사용 설명서

안녕하세요! 이 프로그램은 네이버 카페에 글을 자동으로 써주는 똑똑한 도구입니다.  
개발 지식이 없어도 쉽게 사용할 수 있도록 대시보드(제어판)를 만들었습니다.

---

## 🚀 1. 시작하는 법

1. **프로그램 켜기**
   - 바탕화면이나 폴더에서 `run.py`를 실행하거나, 터미널에 `python run.py`를 입력하세요.
   - 검은색 창(서버)이 켜지고, 잠시 후 인터넷 창으로 **"대시보드"**가 뜹니다.
     - (안 뜨면 크롬 주소창에 `http://127.0.0.1:8000` 입력)

2. **메인 화면 (Dashboard)**
   - **Accounts**: 네이버 계정을 연결하고 관리하는 곳입니다.
   - **Tasks**: 어떤 글을 쓸지(제목, 내용, 게시판) 설정하는 곳입니다.

---

## 🔑 2. 계정 연결하기 (Accounts)

처음 한 번은 로그인을 해줘야 봇이 그걸 기억합니다.

1. **Connect Account** 버튼 누르기.
2. 경고창이 뜨면 **"확인"**을 누르세요.
3. **새로운 크롬 창**이 뜰 겁니다.
4. 거기서 **네이버 로그인**을 직접 해주세요. (평소 하듯이)
5. 로그인이 완료되면 창이 자동으로 꺼지거나, 대시보드 새로고침 후 **"Verify Status"** 버튼을 눌러보세요.
   - **"Logged In"** (초록색)이 뜨면 성공!

---

## 📝 3. 자동 글쓰기 작업 만들기 (Tasks)

1. **New Task** 버튼 클릭.
2. 빈칸 채우기:
   - **Task Name**: 작업 이름 (예: `매일 아침 인사글`)
   - **Account**: 방금 로그인한 계정 선택.
   - **Board URL**: 글을 쓸 카페 게시판 주소.
     - *팁: 그냥 카페 메인 주소나, 아무 게시판 주소를 넣어도 알아서 찾아갑니다.*
   - **Board Name**: (선택사항) 게시판 이름. 비워둬도 됩니다.
   - **Content**: 본문 내용.
3. **Create Task** 클릭.

---

## ▶️ 4. 실행하기 (Run Now)

1. 리스트에 생긴 작업의 **"Run Now"** 버튼을 누르세요.
2. **봇이 작동하는 브라우저 창**이 뜹니다.
3. 봇이 알아서:
   - 카페로 이동하고
   - 글쓰기 버튼을 찾고
   - 제목과 내용을 쓰고
   - "등록" 버튼을 누릅니다.
4. 모든 게 끝나면 창이 닫힙니다.

---

## ❓ 자주 묻는 질문 (FAQ)

**Q. "글쓰기" 버튼을 못 찾는대요!**  
A. 봇이 모바일 주소인지 PC 주소인지 헷갈려서 그럴 수 있습니다. 최신 버전에서는 봇이 알아서 "PC 화면"으로 바꿔서 접속하므로 대부분 해결됩니다.

**Q. 로그인이 자꾸 풀려요.**  
A. **"Verify Status"** 버튼을 눌러 상태를 확인하세요. 만약 "Logged Out"이 뜨면 **Connect Account**로 다시 로그인해주세요.

**Q. 봇이 쓴 글을 지우고 싶어요.**  
A. 아직 "글 삭제" 기능은 개발 중입니다. 카페에 가서 직접 지워주세요!

- 너무 짧은 시간에 글을 많이 쓰면 네이버가 차단할 수 있으니 주의하세요.
